Pregunta 4 del kahoot 1.3

Explicación de las rutas

  • Barra "/" (ruta absoluta): hace referencia a la raíz del proyecto

  • Punto barra "./" (ruta relativa): hace referencia al fichero actual o a la carpeta actual (es lo mismo)

  • Punto punto barra "../" (ruta relativa): hace referencia a la carpeta madre del fichero o carpeta actual
